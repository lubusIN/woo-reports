== Installation ==

***From your WordPress dashboard***
 1. Visit 'Plugins > Add New'
 2. Search for `woo-reports`  or upload zip file
 3. Activate `woo-reports` from your Plugins page

***Manual Installation***
 1. [Download](https://wordpress.org/plugins/woo-reports/) "woo-reports".
 2. Upload the `woo-reports` directory to your '/wp-content/plugins/' directory, using your favorite method (ftp, sftp, scp, etc...)
 3. Activate `woo-reports` from your Plugins page.
